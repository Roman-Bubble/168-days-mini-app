# 168 DAYS Mini App Prototype

A clickable, mobile-first frontend prototype for the **168 DAYS** Telegram Mini App. It presents a premium dark gamified experience for an interactive startup reality show where viewers vote on startup decisions, earn XP, view startup cards, and manage their profile.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React local state
- Mock data only

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open the local URL printed by Next.js in your browser. On desktop, the mobile app is centered in a dark frame around 390–430px wide.

## Prototype note

This is a frontend-only Telegram Mini App prototype. It does **not** include a backend, Supabase, Telegram Bot API integration, or payments. All data lives in `src/data/mock.ts`.
